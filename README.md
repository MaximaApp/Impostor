# 🕵️‍♂️ Impostor - A Party Game

Impostor is a modern, mobile-optimized social deduction party game built to run entirely in your browser. Gather your friends around a single device, select a category, and try to hunt down the network anamoly, or the Impostor, before they decipher the secret word!

If players would like a twist to the game, the Impostor can be given an **ultra-broad Hint** that is somewhat connected to the word to prevent instant giveaways while also forcing them to blend in utilizing generic logic.

---

## ✨ Features

- **📱 Single-Device Local Play:** Perfect for parties. Pass the phone around to receive your secret clearances securely.
- **🧠 Balanced Safe-Hint Engine:** Built-in safeguards replace dead-giveaway hints with highly generic structural industry terms.
- **🎲 Randomized Starting Responsibilities:** At the start of each round, a secure algorithm assigns clue-giver priority to a crewmate to instantly kick off discussions.
- **🌓 Adaptive Interface UI:** Toggle between a variety of unique palette themes seamlessly with just one click.
- **⚙️ Dynamic Clue Controls:** Easily switch the Impostor's clue engine on or off directly from the website's settings.
- **✨ Fluid Animations:** Built with modern CSS transitions and Tabler-icons for a smooth, high-fidelity experience.
