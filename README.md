# 🕵️‍♂️ Impostor - Party Game

A modern, mobile-optimized social deduction party game built to run entirely in the browser. Gather your friends around a single device, select a category, and try to hunt down the network anomaly (Impostor) before they decipher your secret word!

The twist? The Impostor is given an **ultra-broad, single-word structural anchor** (also known as a Hint) to prevent instant giveaways while forcing them to blend in utilizing generic logic.

---

## ✨ Features

- **📱 Single-Device Local Play:** Perfect for parties. Pass the phone around to receive your secret clearances securely.
- **🧠 Balanced Safe-Hint Engine:** Built-in safeguards replace dead-giveaway hints with highly generic structural industry terms.
- **🎲 Randomized Starting Responsibilities:** At the start of each round, a secure algorithm assigns clue-giver priority to a crewmate to instantly kick off discussions.
- **🌓 Adaptive Interface UI:** Toggle between crisp light mode and deep cyber-dark mode themes seamlessly.
- **⚙️ Dynamic Clue Controls:** Easily switch the Impostor's clue engine on or off directly from the lobby.
- **✨ Fluid Animations:** Built with modern CSS transitions and Tabler-icons for a smooth, high-fidelity experience.
